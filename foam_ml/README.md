# `foam_ml` package

Machine-learning analysis for the single-column foam robot.

---

## Current files

| File | Description |
|---|---|
| `foam_ml/north_height_model.py` | Models the relationship between north displacement, foam height, and motor positions (trains Linear Regression + MLP in four directions) |
| `foam_ml/north_trajectories.py` | Visualises all `move_N_` trajectories from training data (top-down, side, 3D views, and motor panels) |

Both scripts are standalone (no ROS environment needed) and load training data automatically by walking up from the working directory to find `actuator/data_collection/training_data/`.

---

## Running the analysis scripts

```bash
# From anywhere inside the workspace
python3 src/foam_ml/foam_ml/north_height_model.py
python3 src/foam_ml/foam_ml/north_trajectories.py
```
